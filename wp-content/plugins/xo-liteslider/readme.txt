== Description ==

XO Slider is a plugin that allows you to easily create sliders.

= Feature =

* Supports thumbnail navigation, carousel, and content.
* Swiper is used for the slider script.
* jQuery is not used on the front side.
* Supports video (Attachment file & YouTube).
* The template file allows detailed customization.

== Installation ==

1. Upload the `xo-liteslider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
